Upcoming Games



Ping

Ping is the latest multiplayer skill based game from EAUG. Both players have control of a paddle that they can move up and down to try and stop the ball from getting past.The Games expected release is in late November.



Space Hallway

Space Hallway is the hyped sequal to the exceptional Hallway Shooter series. The game is similar to the rest of the Hallway Shooter series but with new features and far better graphics. The Games expected release is in late November.



The quest of the rainbow horse

The quest of the rainbow horse is the sequal to the critically acclaimed weird horse. The game has the same unique theme but with a story line added. The Games expected release is in late November.
